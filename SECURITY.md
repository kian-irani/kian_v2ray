# امنیت — KIAN V2Ray

## این ریپو عمومی است و هیچ راز/کلیدی ندارد

این مخزن **هیچ‌کدام** از این‌ها را شامل نمی‌شود و نباید شامل شود:
- توکن GitHub / Cloudflare / تلگرام
- رمز یا کلید SSH سرور
- آی‌پی سرورِ مشخص (کاربر آی‌پی خودش را در صفحه وارد می‌کند)
- کلید خصوصی Reality یا UUID مشخص
- شمارهٔ کارت / آدرس کیف‌پول

## کلیدها کجا ساخته می‌شوند؟

- کلید **X25519** برای Reality و همهٔ **UUID**ها در **مرورگرِ خودِ کاربر** ساخته می‌شوند
  (با `tweetnacl` لوکال؛ بدون هیچ درخواست شبکه‌ای).
- این کلیدها فقط داخل **دستور نصب** قرار می‌گیرند که کاربر **خودش** روی سرورِ **خودش** اجرا می‌کند،
  و در `‎/etc/xray/config.json` (با `chmod 600` روی `users.json`) ذخیره می‌شوند.
- هیچ کلیدی به سرورِ سازنده/شخص ثالث ارسال نمی‌شود.

## توصیه‌ها

- اگر هر وقت یک توکن/رمز به‌اشتباه جایی نشت کرد (مثلاً در یک فایل یا چت)، **همان لحظه Revoke/Roll کن**.
- فایل‌های حاوی راز (مثل نالج داخلی، `users.json`، `‎.env`) را هرگز کامیت نکن — به [.gitignore](.gitignore) نگاه کن.
- بعد از راه‌اندازی، اگر روی سرور مشترک کار می‌کنی، دسترسی فایل‌ها را محدود نگه دار.

## گزارش آسیب‌پذیری

اگر مشکل امنیتی پیدا کردی، لطفاً به‌صورت خصوصی از طریق Issues (بدون افشای جزئیاتِ بهره‌برداری) یا کانال پروژه اطلاع بده.
