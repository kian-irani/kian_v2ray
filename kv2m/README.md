## Kv2m چه می‌کند؟

همان کارهای صفحهٔ تعاملی، ولی **مستقیم از روی کامپیوتر/گوشی و خیلی کامل‌تر** — بدون اینکه دستی وارد ترمینال شوی:

- 🔌 خودش به سرور **SSH** می‌زند (با رمز یا کلید).
- 🚀 **نصب/ساخت کانفیگ تازه**: کلید Reality را همان‌جا می‌سازد، کانفیگ چند-SNI/چند-پورت + WARP + حالت بدون SNI را تولید و روی سرور نصب می‌کند.
- 👥 **مدیریت کاربر**: افزودن، حذف، تمدید، تغییر/صفر کردن حجم — با یک کلیک.
- 📊 **داشبورد**: وضعیت سرویس + جدول کاربرها با مصرف و انقضا.
- 🔗 **کانفیگ‌ها**: گرفتن لینک‌های هر کاربر و کپی یک‌جا.
- 🧩 با Xrayِ موجود **تداخل ندارد** (مسیر و کانتینر ایزوله؛ گزینهٔ «پورت پایه»).

> Kv2m فقط روی سروری کار می‌کند که قبلاً نصبِ kian-v2ray روی آن انجام شده، یا از خود Kv2m (تب «نصب تازه») نصبش کنی.

## 💻 اجرا روی کامپیوتر (ویندوز / مک / لینوکس)

نیاز: **Python 3.9+** (روی ویندوز هنگام نصب، تیک «Add Python to PATH» را بزن).

```bash
# ۱) دانلود پوشهٔ kv2m
git clone https://github.com/KIAN-IRANI/kian_v2ray
cd kian_v2ray/kv2m

# ۲) نصب وابستگی‌ها
pip install -r requirements.txt

# ۳) اجرا (رابط گرافیکی)
python kv2m.py
```

برای رابط خط فرمان (سبک‌تر): `python kv2m.py --cli`

### ساخت فایل اجراییِ ویندوز (.exe) — اختیاری
تا کاربر بدون نصب پایتون فقط روی فایل دوبار کلیک کند:

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name Kv2m kv2m.py
# خروجی در پوشهٔ dist/Kv2m.exe
```

## 📱 اجرا روی اندروید (با Termux)

۱. **Termux** را از [F-Droid](https://f-droid.org/packages/com.termux/) نصب کن (نسخهٔ Play Store قدیمی است).
۲. در Termux:

```bash
pkg update -y && pkg install -y python git
git clone https://github.com/KIAN-IRANI/kian_v2ray
cd kian_v2ray/kv2m
pip install -r requirements.txt
python kv2m.py --cli
```

> روی اندروید، حالت **CLI** اجرا می‌شود (منوی متنی ساده). نسخهٔ گرافیکیِ بومی اندروید (APK) در نقشهٔ راه است.

## 🔐 امنیت

- رمز/کلید SSH فقط روی دستگاهِ خودت می‌ماند و **هیچ‌جا ذخیره یا ارسال نمی‌شود**.
- کلیدهای Reality همان لحظه روی دستگاه تو ساخته می‌شوند.
- ارتباط فقط بین دستگاه تو و سرور خودت است (SSH مستقیم).
